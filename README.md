My Dotfiles

.emacs.d
------
+ Key bindings 
+ Japanese
+ Ruby
+ javascript
+ PHP
+ Haskell
+ Scala

Usage:
```
$ cd $HOME
$ ln -s /path-to/github/dotfiles/.emacs.d .
```
